## 20组"课本英语 vs 自然英语"对照表

| 教科书英语 | 自然地道表达 | 使用场景 |
|-----------|-------------|----------|
| How are you? | How's it going? / How have you been? | 日常打招呼 |
| I'm fine, thank you. | I'm good, thanks. / Not bad! | 回应问候 |
| I'm busy. | I'm swamped. / I've got a lot on my plate. | 工作/学业繁忙 |
| Can you help me? | Could you give me a hand? / Can you do me a favor? | 请求帮助 |
| I'm tired. | I'm exhausted. / I'm wiped out. | 表达疲劳 |
| I'm hungry. | I'm starving. / I could eat a horse. | 表达饥饿 |
| I don't know. | No clue. / Beats me. / I have no idea. | 表示不知道 |
| I agree. | I'm with you. / Totally. / Makes sense. | 表示同意 |
| Please wait. | Hang on. / Just a sec. | 请稍等 |
| I understand. | Got it. / Makes sense. / I see what you mean. | 表示理解 |
| What's wrong? | What's up? / What's going on? / You okay? | 关心他人 |
| I'm sorry. | My bad. / Sorry about that. / My apologies. | 道歉 |
| Good luck! | You've got this! / Break a leg! / Fingers crossed! | 祝福好运 |
| Nice to meet you. | Great to meet you! / Lovely to finally meet you. | 初次见面 |
| Goodbye. | See you! / Catch you later! / Take care. | 道别 |
| I don't like it. | It's not my cup of tea. / Not really my thing. | 表达不偏好 |
| You're welcome. | No problem. / Anytime. / Don't mention it. | 回应感谢 |
| It's expensive. | It's pricey. / It costs a fortune. / It's a rip-off. | 谈论价格 |
| Calm down. | Take it easy. / Chill out. / Don't sweat it. | 劝人放松 |
| I'll try. | I'll give it a shot. / Let me see what I can do. | 表示尝试 |

## 为什么自然英语更受欢迎？

1. **更短更直接** — 母语者偏爱简短高效的表达
2. **更有感情色彩** — "I'm starving"比"I'm hungry"更有画面感
3. **建立亲近感** — 使用俚语和缩写让对方感觉你"不像外人"
4. **体现文化理解** — 知道什么时候用"Break a leg"而不用"Good luck"说明你懂语境

---

## 练习方法

每天选1组场景练习：
- 早上问好用 "How's it going?" 替代 "How are you?"
- 感谢用 "No problem" 替代 "You're welcome"
- 道歉用 "My bad" 替代 "I'm sorry"
