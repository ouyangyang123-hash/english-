# 英语口语自学系统 · 完整指南

## 系统概述

本系统每天自动从 YouTube/B站抓取英语 Vlog 字幕，利用 AI 提取实用词汇和句型，推送到飞书消息和 Notion 词汇银行，让你在碎片时间中持续积累地道英语表达。

## 学习闭环

```
输入（每天自动）          处理（AI分析）           输出（到你手中）
─────────────────      ────────────────      ─────────────────
YouTube Vlog字幕   →    DeepSeek AI      →   飞书卡片消息（早上6:30）
B站Vlog字幕（备用）→    提取B1+词汇/句型  →   Notion词汇银行（可复习）
                                                     ↓
                                              本地 vlog文稿/ 存档
```

## 两条管线的区别

| 特性 | YouTube 管线（主力） | B站 管线（备用） |
|------|---------------------|-------------------|
| 脚本 | `standalone_report.py` | `daily_vlog_fetch.py` |
| 运行环境 | GitHub Actions (美国) | Windows 本地 |
| 定时 | 每天 22:00 UTC | 需配合 setup_task.ps1 |
| 内容类型 | 英语母语者Vlog | 英语教学视频 |
| 字幕质量 | YouTube 自动生成（较好） | UP主手动/CC字幕（更好） |

## 每天你会收到什么

### 飞书卡片消息（早上 ~6:30）
- 5个Vlog的词汇表和地道句型
- 可直接在手机飞书上阅读学习

### Notion 词汇银行
- 每个生词/短语一条数据库记录
- 字段：单词、中文释义、例句、话题分类、词性、掌握状态
- 支持间隔复习（复习次数、简易度EF、间隔天数）

### 本地存档
```
vlog文稿/2026-05-29/
├── vlog_01.md         # 视频字幕原文
├── vlog_02.md
├── ...
└── report_2026-05-29.md   # 综合学习报告
```

## 如何使用

### 日常学习
1. 早上查看飞书卡片，花5分钟过一遍新词汇
2. 感兴趣的词点击 Notion 链接（如果配置了）标记练习

### 复习
1. 在 Notion 词汇银行中按"掌握状态"筛选"学习中"的词汇
2. 用原文例句做跟读/造句练习
3. 掌握后标记为"已掌握"

### 回看历史
- 所有报告保存在 `vlog文稿/` 目录下
- 可用 `sync_to_notion.py YYYY-MM-DD` 重新导入任何一天的报告

## 扩展与自定义

### 添加搜索源（YouTube）
编辑 `standalone_report.py` 的 `main()` 函数中的 `queries` 列表：
```python
queries = [
    "english vlog daily life",
    "day in my life english vlog",
    # 添加你感兴趣的关键词
    "english tech review vlog",
]
```

### 添加搜索源（B站）
编辑 `daily_vlog_fetch.py` 的 `main()` 函数中的 `keywords` 列表：
```python
keywords = [
    "英语Vlog日常",
    "English vlog 英语学习",
    # 添加你感兴趣的关键词
]
```

### 更换 AI 模型
设置环境变量 `DEEPSEEK_MODEL` 或在 `analyze_subtitle()` 中修改模型名。

### 更换推送渠道
- 飞书：修改 `send_to_feishu.py` 中的 APP_ID / APP_SECRET / OPEN_ID
- Notion：设置 `NOTION_API_KEY` 和 `NOTION_DATABASE_ID` 环境变量
