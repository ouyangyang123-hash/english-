# 英语口语自学系统 · 自动化配置

## 系统状态：✅ 已部署

## 自动化流程

```
每天早上 6:03
     ↓
Claude Code 定时触发
     ↓
WebSearch 搜索英语Vlog文稿 (4篇)
     ↓
WebFetch 抓取文章内容
     ↓
Claude 分析提取词汇/句型
     ↓
  ├── 📱 send_to_feishu.py → 飞书个人消息
  ├── 📊 sync_to_notion.py → Notion 词汇银行
  └── 📁 vlog文稿/{日期}/ → 本地存档
         ├── vlog_01.md ~ vlog_04.md
         └── report_{日期}.md
```

## 文稿存储位置

```
助理团队/
└── vlog文稿/
    ├── 2026-05-28/
    │   ├── vlog_01.md    ← 原始文稿
    │   ├── vlog_02.md
    │   ├── vlog_03.md
    │   ├── vlog_04.md
    │   └── report.md     ← 当日分析报告
    ├── 2026-05-29/
    │   └── ...
```

> 所有文稿永久保存在本地，可随时回看、导入 Notion 复习。
