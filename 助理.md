# 英语口语自学系统 · 自动化配置

## 系统状态：✅ 已部署（双管线运行）

## 自动化流程

### 管线一：YouTube（主力 · GitHub Actions）
```
每天 22:00 UTC (北京时间 6:00 AM)
         ↓
GitHub Actions 触发
         ↓
yt-dlp 搜索 YouTube 英语Vlog (5个视频)
         ↓
下载英文字幕 (.srt)
         ↓
DeepSeek API 提取词汇/句型
         ↓
     ├── 📊 Notion 词汇银行（自动同步）
     ├── 📱 飞书卡片消息
     └── 📁 vlog文稿/{日期}/ → git push 回仓库
```

### 管线二：微信公众号（备用 · Windows 本地）
```
Windows 定时任务 (6:00 AM) / 手动执行
         ↓
daily_vlog_fetch.py → 搜狗/Bing 搜索微信公众号文章
         ↓
搜索英语Vlog/口语学习文章 → 抓取正文内容
         ↓
     └── 📁 vlog文稿/{日期}/ → 本地存档
         （分析/推送可复用管线一的工具）
```

## 快捷命令

```bash
# 完整管线（本地运行，需配置环境变量和代理）
python standalone_report.py

# 微信公众号抓取备用管线
python daily_vlog_fetch.py                              # 搜索并抓取
python daily_vlog_fetch.py --url <微信文章链接>          # 抓取指定文章
python daily_vlog_fetch.py --urls urls.txt              # 批量从文件导入

# 手动同步报告到 Notion
python sync_to_notion.py 2026-05-29

# 手动发送报告到飞书
python -c "from send_to_feishu import send_card; send_card('Title', open('vlog文稿/2026-05-29/report_2026-05-29.md', encoding='utf-8').read()[:4800])"

# GitHub Actions 手动触发
# 访问 https://github.com/ouyangyang123-hash/english-/actions 点击 "Run workflow"
```
